| ---- | ------- |

| npm/npx | 6.14.8 |
| n | 6.7.0 |

# 手順

1. GitHub にリポジトリを作成
2. Zenn と GitHub リポジトリを連携
3. Zenn CLI をインストール
4. ローカルリポジトリの Zenn 用セットアップ
5. Zenn CLI で記事を作成
6. GitHub リポジトリを更新
